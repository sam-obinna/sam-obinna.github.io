Problem

The firm was missing early-stage project opportunities due to manual monitoring of multiple sources (universities, municipalities, news) being time-consuming and inconsistent.

Solution

Built an N8N workflow automation system with LLM-powered agents (Gemini Flash 2.0) to extract project data from PDFs, websites, and documents. Created 4 AI agents for different source types (municipal, institutional, RFP databases, news) and implemented Airtable for centralized data storage with automated prioritization.
