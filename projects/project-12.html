Problem

No way to connect Bokun (booking management platform) with Softr (website builder). Complex OAuth 2.0 authentication blocked direct integration. No customer-facing booking portal existed.

Solution

Custom OAuth middleware (Next.js on Vercel) acting as authentication bridge: Softr frontend to OAuth middleware (handles token storage via Vercel KV, HMAC signatures, API proxy endpoints, GraphQL queries) to Bokun API. Built Softr customer-facing booking portal integrated with middleware + Airtable database.
