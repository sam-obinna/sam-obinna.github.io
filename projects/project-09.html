The Problem

Manual order processing, sales rep assignment, and data cleaning between Shopify and Monday.com were causing fulfilment delays and errors. The team was manually handling 3,000+ orders with 26,000+ line items - an unsustainable workload that was creating inconsistencies in customer-to-rep assignment and holding back revenue growth.

⚙️

The Solution

Built a Zapier real-time pipeline that routes Shopify orders through automated customer-to-rep matching and creates per-rep Monday.com boards instantly. Python scripts (Google Colab) handled bulk historical backfilling of 26K+ line items. A data cleaning pipeline tackled duplicate customer IDs with conservative auto-resolution and manual review flags. Per-rep board architecture with archiving kept the system within Monday.com subscription limits while scaling cleanly.
