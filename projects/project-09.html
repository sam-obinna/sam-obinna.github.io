Problem

Manual order processing, sales rep assignment, data cleaning between Shopify and Monday.com. Inconsistent data caused fulfillment delays and errors. Processing 3,000+ orders with 26,000+ line items manually.

Solution

Zapier real-time pipeline: Shopify orders to automated customer-to-rep matching to Monday.com board creation per rep. Python scripts (Google Colab) for bulk historical backfilling. Data cleaning pipeline handling duplicate customer IDs with conservative auto-resolution + manual review flags. Per-rep board architecture with archiving to stay within Monday.com subscription limits.
