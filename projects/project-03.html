Problem

Manual police report data entry into Clio taking 2+ hours per case, losing clients to competitors with faster turnaround. Paralegals manually typed every detail from scanned PDFs, creating speed-to-lead problems.

Solution

Built n8n workflow with Gemini for OCR, ChatGPT for structuring, Clio API for retainer auto-generation + personalized client emails with seasonal scheduling logic. System reads police reports, extracts information, updates Matters in Clio, generates retainer agreements, and sends personalized emails with booking links.
