MULTI-PAGE PORTFOLIO STRUCTURE
================================

This portfolio consists of 13 HTML files + 1 shared CSS file:

FILES:
------
- index.html          Main landing page with all 12 project cards
- project-01.html     Canadian Architecture Firm
- project-02.html     Government Training Program  
- project-03.html     Personal Injury Law Firm
- project-04.html     Insurance Marketing Agency
- project-05.html     UK Motorcycle Storage Company
- project-06.html     US Music Festival Company
- project-07.html     Ocean Cleanup Nonprofit
- project-08.html     Danish Specialty Coffee Company
- project-09.html     B2B Wholesale Jewelry Company
- project-10.html     Digital Health Startup
- project-11.html     Washington Property Management Company
- project-12.html     Japan Hospitality Company
- styles.css          Shared stylesheet for all pages

FEATURES:
---------
✓ Click any project card on index.html to see full details
✓ Next/Previous navigation on each project page
✓ Back to Portfolio link on every project
✓ Footer navigation showing adjacent projects
✓ Responsive design (mobile, tablet, desktop)
✓ Print-optimized for PDF export
✓ Dark mode support (auto-adapts to system)
✓ Filter projects by industry on index page

SETUP:
------
1. Place all HTML files in the same directory
2. Place styles.css in the same directory
3. Create a "tech/" subfolder for your SVG icons and images
4. Open index.html in a browser to start

NAVIGATION FLOW:
----------------
Index → Project 01 → Project 02 → ... → Project 12 → Back to Index

Each project page has:
- Previous/Next buttons (top right)
- Footer with large Previous/Next cards
- Back to Portfolio link (top left)

All company names are anonymized.
No em dashes used anywhere.
Theme matches your portfolio site (blue/purple gradients).
